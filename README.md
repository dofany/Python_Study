# **Jupyter Notebook을 이용한 Python공부** 

## 1. 변수의 정의 : 임시 기억(저장) 공간 
   + 데이터를 하나만 저장할 수 있다.

## 2. 변수의 특징
   + 묵시적 선언
   + 동적 바인딩
   + immutable(불변성)
    - 변수에 저장되어 있는 값이 변하는게 아니라 데이터가 저장되어있는 주소값이 바뀌는 거라서 불변성이다.   

## 3. Data Type
   + 종류
    
1. 기본 자료형 : 데이터가 하나일 때 사용 
 * int : 정수형
 * float : 실수형
 * str : 문자형
 * boolean : 참/거짓
        
2. 집합 자료형 : 데이터가 여러개 일 때 사용
---
* list : 리스트
  - '[ ]' 사용
  - 임의의 객체를 순차적으로 저장하는 집합적 자료형(Sequence Data)
  - index 사용
  - mutable(변경 가능)
 ---
* tuple : 튜플
  - '( )' 사용
  - immutable(변경 불가)
 ---
* dict : 두 개의 데이터를 하나로 묶음
  - 순서가 없다.
  - {키 : 값}
  - 키를 이용해서 값을 조회
 ---
* set : 원소를 얻고자 사용
  - { } 사용
  - 키가 없고 값을 사용
  - 순서가 없다(인덱스가 없다)
  - 중복 불가
## 4. Operator

   1) 산술 연산자
        *  /  +  -  %(나머지)  //(몫)  **
    
   2) 비교 연산자
        >  <  >=  <=  !=  ==
        
   3) 논리 연산자(단락회로 연산자)
        and(&)  or(|)  not(!)(단항, 토글)  
        
        x /  y  / x and y /  x or y  / not x  / not y  / x xor y(데이터 암호화/복호화)
        
        T /  T  /    T   /    T   /    F   /    F   /    F
        
        T /  F  /    F   /    T   /    F   /    T   /    T
        
        F /  T  /    F   /    T   /    T   /    F   /    T
        
        F /  F  /    F   /    F   /    T   /    T   /    F
    
   4) 대입(치환, 할당) 연산자
        =  +=  -=  *=  /=  ...
    
   5) 포함 여부 연산자
        in,  not in 
    
   6) 비트(이진수) 연산자 
        bit < Byte(8bit) < KB(1024byte) < MB(1024KB) < GB(1024MB)
        
        &(and)  |(or)  ^(xor)  ~(not)  >>(Right Shift)  <<(Left Shift) ...
